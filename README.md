# Swell / Next.js App

Learn how to create a headless ecommerce application using Swell and Next.js

## Built With

- [Swell](https://swell.is/)
- [Tailwind CSS](https://tailwindcss.com)
- [Next.js](https://nextjs.org)
- [Vercel](https://vercel.com)
